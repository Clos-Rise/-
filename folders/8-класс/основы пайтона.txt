  Переменные :

Переменные - так скажем хранилища которые могут хранить определенные типы данных и их значения , например :

a = 5 
тут мы присваиваем переменной a , значение 5 - что является INT(INTEGER), вот крч типы данных которые понадабятся новичкам:

INTEGER - Целые числа
FLOAT - Вещественные числа
STRING - Строки (пример - "привет")
BOOLEAN - True / False
LIST - Изменяемый список элементов
TUPLE - факттически то же что и list но неизменяемый (кортеж)

пример переменных указанных выше (по порядку):

a = 5
b = 5.1
c = "Hello"
d = True
e = [1, 2, 3]
f = (1, 2, 3)

  Базовые Функции:

print() - выводит str или значение переменных на экран
input() - пользовательский ввод значения
len() - выводит кол-во элементов в объекте
type() - возврат типа объекта
max() и min() - возвращает наибольшее или наименьшее значение в объекте
round() - округляет число до тех знаков , которое вы указали.
help() - полезная функция , она выводит справку позволяя вам почитать некую документацию.

  Примеры программ:

Простой калькулятор - что такое калькулятор? В нашем случае - это программа которая считает арифметические выражения над числами, вот наипростейший пример калькулятора (не считая exec()) :

a = 5
b = 5

print(a+b)

тут мы указываем переменные и их значения , а в функции print - складываем оба значения.

надо бы улучшить :

a = float(input())
b = float(input())

print(a+b)

Тут все просто мы указываем на то , что все данные которые мы введем в input() - будут float то есть вещественными , а далее просто прибавляем через print , но надо сделать еще лучше :

a = float(input("Введите 1 число:"))
b = float(input("Введите 2 число:"))

op = input("Введите оператор (+ -) : ")

if op == '+':
    print(a + b)
elif op == '-':
    print(a - b)
else:
    print("Ошибка!")

тут мы добавили else if и elif , а что это? Ща напишу :

IF - дословно ЕСЛИ , то есть ЕСЛИ переменная = 5 то print("Это пять!") // что то на псевдо языке
ELIF - Иначе если , то есть ИНАЧЕ ЕСЛИ переменная != 5 , print("Не пять")
ELSE - Иначе , то есть если ни один блок кода не True , тогда мы выводим ответ .

IF выполняется первым блоком кода .
ELIF - выполняется если ПРОШЛЫЙ блок кода неверный или не соответствует правде.
ELSE - если ВСЕ ПРОШЛЫЕ блоки кода неверные , выполняем действие.
